Download JBang

Note: Java is not required to be installed - JBang will automatically download a Java version
if no Java 8 or higher Java Development Kit (JDK) version is found.

  • Windows
  • Linux
  • Mac
  • Docker
  • GitHub Actions
  • Maven
  • Gradle

curl/iex

Powershell: 

iex "& { $(iwr https://ps.jbang.dev) } app setup jbang"

GitBash/cygwin/mingwin/WSL: 

curl -Ls https://sh.jbang.dev | bash -s - app setup jbang

Chocolatey

choco install jbang

Scoop

scoop bucket add jbangdev https://github.com/jbangdev/scoop-bucket
scoop install jbang

Manual

Unzip the latest binary release, put the jbang-[version]/bin folder in to your $PATH and you are set.

Zero install

You can run download and run jbang directly using Powershell with iex "& { $(iwr https://ps.jbang.dev) } [jbang arguments]" or if you have bash based shell/terminal curl -Ls https://sh.jbang.dev | bash -s - [jbang arguments]
